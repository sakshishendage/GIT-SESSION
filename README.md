# Book Store Management System

A Python-based Book Store Management System to manage inventory, sales, and book records.

## Features
- Add new books to the store
- Manage inventory (update/delete books)
- View book details
- User-friendly console interface

## Files
- `main.py` → The main program to run the system
- `operations.py` → Functions for book operations (add, update, delete)
- `validation.py` → Helper functions to validate input data

## Technologies Used
- Python 3
- IDLE (Python IDE)

## How to Run
1. Clone the repository or download the files.
2. Open `main.py` in Python IDLE or any Python IDE.
3. Run the program and follow the on-screen instructions.
